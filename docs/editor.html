Файл — для машины,
интерфейс — для человека.

Расширение для VS Code и Cursor превращает физический .xc-файл в удобный сплит-вью: слева редактируемый монолитный код, справа — смысловой слой в одном из двух режимов.

auth.xc

Сетка — строки справа выровнены со строками кода 1:1; блок занимает строки до начала следующего, клик раскрывает его целиком, несколько блоков на одной строке становятся вкладками. Контекст — карточка описания появляется при наведении на строку кода. Наведите курсор на код или на карточку — связанные строки подсветятся.

Возможности

Чтение и авторство в одном месте.

🧩

Два режима отображения

  • Сетка (Excel-style): прокрутка монолитна — единый индекс строк, рассинхрон невозможен
  • Сворачивание блока в одну строку и вкладки при совпадении начала
  • Контекст: карточки только для строки под курсором
✍️

Авторство

  • Правка блока — id, диапазон строк и Markdown — во всплывающем редакторе
  • Описать выделение: привязка прозы к выделенным строкам кода
  • Удаление блока с Ctrl/Cmd+Z
🎨

Богатый рендер

  • Подсветка синтаксиса и номера строк слева
  • Таблицы, чек-листы и формулы LaTeX ($…$ → MathML)
  • Правка кода автоматически сдвигает диапазоны lines:
🔀

Изолированный git-diff

  • XC: Diff Code Layer — только код
  • XC: Diff Explanation Layer — только проза
  • Правки текста не создают шум на код-ревью
Установка

Поставить за минуту.

Собранный пакет лежит в репозитории.

# установить собранный .vsix
code --install-extension \
  vscode-extension/versions/explained-code-2.2.0.vsix --force

# или режим разработки
cd vscode-extension
npm install && npm run compile   # затем F5 в VS Code
.xc Explained Code — открытый формат и инструментарий.
